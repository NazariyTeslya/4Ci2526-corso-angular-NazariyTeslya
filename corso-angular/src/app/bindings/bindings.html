String interpolation

Con la string interpolation visualizzo nel template HTML un valore memorizzato in una "variabile" Typescript.

Possono essere usate solo variabili che sono attributi della classe component.

In una applicazione zoneless, per garantire che venga aggiornata, devo usare i signals.

Un signal è una funzione wrapper che gestisce un valore di qualsiasi tipo.

Il signal controlla se un valore è stato modificato, marca la zona come "dirty" e chiede il refresh del template HTML.

  1. Dichiarare il signal in Typescript
  2. Inserire una chiamata alla funzione signal tra doppie parentesi graffe

Event binding

Per utilizzare event binding, si toglie on dal nome dell'evento e si racchiude fra parentesi tonde il nome rimanente. Il valore dell'evento DEVE essere il nome di un metodo della classe del componente. ATTENZIONE: ricordarsi le parentesi tonde dopo il nome del metodo!!!
